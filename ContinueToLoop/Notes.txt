Using break and continue Statements
In Java, break and continue statements offer powerful control over loop execution. They allow you to alter the standard flow of for, while, and do-while loops, enabling you to skip iterations or exit loops entirely based on specific conditions. Mastering these statements is crucial for writing efficient and tailored loop logic.

The break Statement
The break statement is used to terminate a loop prematurely. When a break statement is encountered inside a loop, the loop's execution is immediately stopped, and the program control transfers to the next statement after the loop.

Basic Usage
The most common use case for break is to exit a loop when a certain condition is met. Consider the following example:

java

public class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                break; // Exit the loop when i is 5
            }
            System.out.println("i = " + i);
        }
        System.out.println("Loop finished.");
    }
}
In this code:

We initialize a for loop that iterates from 1 to 10.
Inside the loop, we have an if statement that checks if i is equal to 5.
If i is 5, the break statement is executed.
The break statement terminates the loop immediately.
The program then proceeds to execute the next statement after the loop, which prints "Loop finished."
The output of this program will be:

javascript

i = 1
i = 2
i = 3
i = 4
Loop finished.
Notice that the loop stops executing when i reaches 5, and the remaining iterations are skipped.

break in Nested Loops
When dealing with nested loops, the break statement only terminates the innermost loop in which it is present. To break out of an outer loop from within an inner loop, you can use labeled break statements.

java

public class NestedBreakExample {
    public static void main(String[] args) {
        outerLoop: // Label for the outer loop
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                if (i == 2 && j == 2) {
                    break outerLoop; // Break out of the outer loop
                }
                System.out.println("i = " + i + ", j = " + j);
            }
        }
        System.out.println("Nested loop finished.");
    }
}
In this example:

We have nested for loops. The outer loop iterates from 1 to 3, and the inner loop also iterates from 1 to 3.
We've labeled the outer loop as outerLoop.
Inside the inner loop, we check if i is 2 and j is 2.
If both conditions are true, we execute break outerLoop;. This statement tells the program to break out of the loop labeled outerLoop.
As a result, when i is 2 and j is 2, both the inner and outer loops are terminated, and the program proceeds to print "Nested loop finished."
The output of this program will be:

javascript

i = 1, j = 1
i = 1, j = 2
i = 1, j = 3
i = 2, j = 1
Nested loop finished.
Without the label, only the inner loop would have terminated when i equaled 2 and j equaled 2.
